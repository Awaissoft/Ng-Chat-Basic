Angular Basic Chat
=============

** Features Include **
* Login/logout
* PubNub messaging (no server or database required)
* Uses bootstrap
* Realtime chatting

Tutorial: http://jadendreamer.wordpress.com/2013/10/17/angular-javascript-chatroom-tutorial-no-database-or-sockets-required/
